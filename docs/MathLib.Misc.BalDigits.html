    ToBalancedDigits(BigInteger, int[])

    Converts a specified integer into a sequence of constrained digits based on the constraints provided in constraints.

    Declaration
    public static int[] ToBalancedDigits(BigInteger integer, int[] constraints)
    Parameters
    Type Name Description
    BigInteger integer

    The target integer value to represent as a sequence of constrained digits.
    int[] constraints

    A list of non-negative integers, each defining constraints for the respective position in the sequence.
    Returns
    Type Description
    int[]

    An array of integers representing integer as a sequence of constrained digits.
    Remarks

    This method generates a sequence representing integer as a weighted sum in base-2, adhering to constraints in constraints. Each constraint specifies:

    • The maximum absolute value of the corresponding digit.
    • The parity requirement (odd/even) based on the parity of the constraint.
    The minimum length of constraints is 3, and the first and last elements are always 1, ensuring that the sequence starts and ends with either -1 or 1.

    The resulting sequence meets the following conditions:

    1. Its length matches the length of constraints.
    2. The weighted sum of the sequence, using powers of 2 (1, 2, 4, 8, ...), equals integer.
    3. Each digit in the sequence meets both the absolute value and parity requirements specified by the corresponding constraint.
    4. The first and last digits are constrained to either -1 or 1.
    Examples
    ToBalancedDigits(1, new int[] { 1, 1 })  => [-1, 1]  //1 = 1 * 2^0 + 1 * 2^1)

ToBalancedDigits(-1, new int[] { 1, 1 }) => [1, -1] //-1 = 1 * 2^0 + (-1) * 2^1)

ToBalancedDigits(-25, new int[] { 1, 2, 3, 2, 1 }) => [-1, -2, -1, 0, -1] //-25 = -1 * 2^0 + (-2) * 2^1 + (-1) * 2^2 + 0 * 2^3 + (-1) * 2^4)
