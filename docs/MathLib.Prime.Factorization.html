    Class Factorization

    Represents the prime factorization of an integer, which may be partial. Handles composite remainder factors and negative integers.

    Inheritance

    Syntax
    public sealed class Factorization
    Remarks

    This class is immutable. The factorization is initialized via a primary constructor that accepts the prime factors and the remainder factor.
